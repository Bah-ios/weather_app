# Weather App
A weather app built with vanilla JavaScript and the OpenWeatherMap API.

## Features
- Live weather data for any city
- Temperature, humidity, wind speed, feels like
- Weather icons
- Error handling
- Enter key support

## Tech Stack
- HTML, CSS, JavaScript
- OpenWeatherMap API
